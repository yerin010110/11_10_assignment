# 🌟 Star Drift (Kirby Edition)

**Pygame 기반 OOP 미니 게임 프로젝트**  
커비를 조작하여 적을 피하고 아이템을 수집하며 최고 점수를 기록하는 **서바이벌 게임**입니다

---

## 🎮 게임 소개

| 항목      | 내용                                                          |
| --------- | ------------------------------------------------------------- |
| 게임 장르 | 2D 아케이드 / 회피 & 생존                                     |
| 테마      | 커비(Kirby)                                                   |
| 조작      | WASD / 방향키                                                 |
| 목표      | 생명을 지키며 최고 점수 달성                                  |
| 특이 요소 | 하트(생명) 시스템, 충돌 무적시간, 재시작 가능, 최고 점수 저장 |

---

## 🧾 주요 기능

✔ OOP 구조 기반 설계 (Game, Player, Enemy, Item, Background)  
✔ 적 충돌 시 **하트(생명 3개) 감소**  
✔ 생명 0이 되면 **게임 오버 화면 출력**  
✔ 아이템 획득 시 점수 증가 또는 체력 회복  
✔ 최고 점수 자동 저장 (`highscore.json`)  
✔ Restart / Quit 버튼 UI 제공  
✔ 충돌 후 **1 5초 무적** 적용  
✔ BGM 및 효과음 지원(파일 없을 시 자동 예외 처리)

---

## 🕹 조작 방법

| 키            | 동작                              |
| ------------- | --------------------------------- |
| 방향키 / WASD | 플레이어 이동                     |
| 마우스 클릭   | 게임 오버에서 Restart / Quit 선택 |

---

## ⚠ 참고 사항

- 사운드 파일이 없어도 게임은 정상 실행됩니다
- 하이스코어는 `highscore.json`에 자동 저장됩니다
- 이미지 파일명은 대소문자를 정확히 지켜야 합니다

---

## 👨‍💻 개발 포인트

- Delta Time을 활용한 프레임 독립 이동 처리
- `pygame.Rect` 기반 충돌 처리 최적화
- 무적시간을 통한 연속 충돌 방지
- 리스트 안전 순회(`for obj in list[:]`)로 삭제 오류 방지
